## Projects to start with

- [myCat](https://github.com/yumiaura/myCat) — a cute Qt desktop cat overlay
  with reminders, local Ollama chat and cross-platform builds.
- [AICommit](https://github.com/yumiaura/AICommit) — local-LLM commit message
  helper for Conventional Commits, reviews and changelog entries.
- [followme](https://github.com/yumiaura/followme) — discover, grade, follow
  and star GitHub repositories with a local Ollama model.
- [SubForge](https://github.com/yumiaura/SubForge) — media to subtitles to
  translation, with CLI, TUI and GUI frontends.
- [audio8bit](https://github.com/yumiaura/audio8bit) — turn songs into
  chiptune-style audio from the terminal.
